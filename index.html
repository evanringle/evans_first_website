Welcome to your Bootstrap Test Project

You will recreate this webpage and an additional Form Page

Just follow the instruction steps here to re-create these webpages!

  • Create a general landing page with a JumboTron
  • Next create a NavBar on the Page that links to another html file
  • Add two paragraphs of lorem-ipsum below the JumboTron
  • Next use the Grid system to add in thumbnail pictures of aircraft. Source: https://stocksnap.io/search/coffee You will need to use the thumbnail class for this. Here are the image links:
    • one
    • two

  • Next Make sure that at the largest setting you have 3 columns of pictures. on the smallest screen settings you should have 2 columns. it's up to you where you want to change
  • Next create another html file for the sign up page
  • Next create a form on the sign up page with email, password, and Check Box

That's it. Feel free to play around with this project and style it more to your liking. The main focuses of this project are the uses of NavBar, Jumbotron, Form, Container, and the Grid System
